# Windows_Lock_Machine_Program  
**描述**：在一段时间内限制你对Windows的访问。

**用法**：1.打开 bin/Debug 文件夹并以管理员身份运行 Windiws 锁机程序.exe  
     2.设置锁机时间（以秒为单位）  
     3.点击“开始锁机”按钮进行锁机。  
	
**注意**：1.若要退出程序，请点击“退出”按钮。“退出”按钮在开始锁机后不可用。  
      2.锁机一旦开始，则不能（强制）终止。请不要尝试在锁机期间进行任何操作，所有在锁机期间尝试的操作只有在锁机结束后才会（立即）生效。
